# 🍊 Fanta Landing Page 
> Projeto de interface de alto impacto visual desenvolvido durante o treinamento **Rei do Código**.

Este projeto consiste em uma landing page moderna da **Fanta**, focada em animações fluidas, design disruptivo e uma experiência de usuário (UX) premium. Desenvolvido para aplicar conceitos avançados de Front-end aprendidos na prática.

---

## 🚀 Tecnologias e Conceitos
Para alcançar este resultado, utilizei:

* <img src="https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white" height="18"> **Estrutura Semântica:** Foco em SEO e organização de código.
* <img src="https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white" height="18"> **Estilização Avançada:** Layouts modernos e cores vibrantes.
* <img src="https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black" height="18"> **Lógica de Interatividade:** Manipulação dinâmica de elementos.
* 🎥 **Animações:** Aplicação de conceitos de *ScrollTrigger* e movimentos fluidos (GSAP).

---

## ✨ Destaques do Projeto
- [x] **Animações de entrada:** Elementos que surgem de forma coordenada.
- [x] **Efeito de Scroll:** Interatividade baseada no movimento do usuário.
- [x] **Assets Otimizados:** Uso de fontes personalizadas e imagens de alta qualidade.
